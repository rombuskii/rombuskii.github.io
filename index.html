Intro

Hey, welcome to my workshop. My name is Eruba Ekedayen, and I'm currently pursuing a Bachelors of Science in Computer Science at Northeastern University. I'm originally from Lagos, Nigeria. Born and raised with my two older brothers. As young as I could remember I've always had a fascination for technology. From back in the day, when Nokia's ruled the world and Bounce or Snake were the hottest games you could download on a phone. Or when my brother brought home a Gameboy Advance for his birthday with Pokemon Yellow and we would all gather arounds sinking countless hours into the game. Over the years, I've dabbled in many fields of computing: Gaming, Web Design, Cybersecurity, Media, Database Design and more. Each experience made my journey learning richer and more exciting. Hopefully, this workshop imparts that same feeling into you. Here's a sample of some awesome stuff I've worked on!

Projects

Machine Learning

As a machine learning enthusiast, I have developed several projects that showcase my skills in data analysis, modeling, and prediction. My portfolio includes projects such as sentiment analysis of social media data using natural language processing techniques, image classification using convolutional neural networks, and anomaly detection in time-series data using unsupervised learning algorithms. These projects demonstrate my proficiency in Python programming, data manipulation with Pandas, data visualization with Matplotlib and Seaborn, and machine learning libraries like Scikit-Learn and TensorFlow. I have also contributed to open-source projects on GitHub and participated in Kaggle competitions to hone my skills and stay up-to-date with the latest trends in machine learning. Through these projects, I have gained a solid understanding of various machine learning techniques and their practical applications, and I am eager to apply my skills to real-world problems.


Java/Data Design

As a Java and Data Design enthusiast, I have developed several projects that showcase my skills in developing robust and scalable software systems, as well as my understanding of data structures and algorithms. My portfolio includes projects such as a data-driven web application using Spring MVC and Hibernate, a high-performance messaging system using Apache Kafka, and an automated trading system using Java and MySQL. These projects demonstrate my proficiency in Java programming, as well as my understanding of design patterns, RESTful APIs, and database design. I have also contributed to open-source projects on GitHub and participated in hackathons to stay up-to-date with the latest trends in Java and data design. Through these projects, I have gained a solid understanding of various software development methodologies and the importance of clean, modular code. I am passionate about designing and building software systems that are not only functional but also maintainable, extensible, and efficient.

Web Design

As a web designer, I am passionate about creating beautiful, user-friendly, and responsive websites that provide a great user experience. My portfolio includes projects such as as interest forms, an e-commerce site for a drone company, and a tribute page. These projects demonstrate my proficiency in web design tools as well as my understanding of front-end development languages and frameworks such as HTML, CSS, JavaScript, and React. I also have experience in creating wireframes, mockups, and prototypes, as well as conducting user testing to improve the usability and functionality of the websites I design. I am constantly learning and keeping up-to-date with the latest design trends, and I am passionate about using design to solve real-world problems and make a positive impact on people's lives. If you're looking for a web designer who can create stunning and intuitive websites, please take a look at my portfolio and feel free to reach out to me.
